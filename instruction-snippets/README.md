# Instruction Snippets

This folder stores reusable instruction-file snippets.

These files are **templates to copy into other repositories or user config**, not active instructions for this repository.

Repo-level snippets:
- `instruction-snippets/AGENTS.md`
- `instruction-snippets/CLAUDE.md`
- `instruction-snippets/GEMINI.md`
- `instruction-snippets/.github/copilot-instructions.md`

See `docs/install-instruction-snippets.md` for installation options.
